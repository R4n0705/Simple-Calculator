# Simple Calculator

A simple web-based calculator built with HTML, CSS, and JavaScript. This calculator supports basic arithmetic operations like addition, subtraction, multiplication, and division. 
It also includes buttons for clearing, deleting, and calculating percentages.

---

## Features

- Addition, subtraction, multiplication, and division
- Clear and delete buttons
- Percentage calculation
- Responsive design for different screen sizes
- Clean and modern interface using CSS variables
